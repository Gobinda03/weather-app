# 🌦 Weather App

A simple and modern **Weather Application** built using **HTML, CSS, and JavaScript** that fetches real-time weather data using the OpenWeather API.

## 🚀 Live Demo

```
https://Gobinda03.github.io/weather-app
```

---

## ✨ Features

* 🔍 Search weather by city name
* ⌨️ Press **Enter** to search instantly
* 📍 Automatically detect user's location
* 🌡 Displays **temperature, humidity, wind speed**
* 🌤 Shows **weather description and icon**
* 🎨 Dynamic background based on weather conditions
* 💎 Modern **glassmorphism UI design**

---

## 🛠 Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript (ES6)**
* **OpenWeather API**

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/Gobinda03/weather-app.git
```

2. Navigate to the project folder

```
cd weather-app
```

3. Open `index.html` in your browser

---

## 🔑 API Used

This project uses the **OpenWeather API** to fetch real-time weather data.

Website: https://openweathermap.org/api

---

## ⚠️ Note

The API key is included in the frontend for demonstration purposes.
For production applications, the API key should be stored securely using a backend server.
